# Core Values

## 1. 品味先于设计

我们不从 Figma 开始。我们从对话开始。

先听。先理解。先品味。

在真正了解一个人之前，不动笔。

**没有品味过的设计，只是装饰。**

## 2. 真实重于完美

一个人的一生不完美。他的数字身份也不应该完美。

保留那些粗糙的、真实的、属于这个人的痕迹。不要用设计去美化它。用设计去呈现它。

**完美是机器做的。真实是人做的。**

## 3. 雕刻需要时间

品味要时间。理解要时间。找到那几个真正属于他的关键词——要时间。

我们不做快餐。我们不给两周的交期。我们不在还没理解一个人的时候就开始设计。

**不急。不赶。品味需要时间。**

## 4. 日常和宏大同样重要

不是只有刻骨铭心的经历才值得被雕刻。

一个普通的星期四下午。
每天早上喝的那杯咖啡。
窗台上那盆养了三年的植物。

**日常里有最深的真相。**

## 5. 一个人的一生不能被模板化

没有两个一样的人。没有两个一样的数字身份。

不要套模板。不要走捷径。每一次都从零开始品味。

**因为每一次，都是一个独一无二的人生。**

## 6. 数字身份是活的

不是交付了就结束了。

一个人的一生在继续。他的数字身份也应该继续。它可以生长，可以变化，可以随着他一起变老。

**我们不是在做一个项目。我们是在陪伴一个人的一生。**

---

*这六条价值观，是 WKALAN 的底线。不是口号。是做事的方式。*
