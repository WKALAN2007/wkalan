# 006 — Layout System

## Philosophy

**Every page tells a story. The layout is the narrative structure.**

Layout is not about arranging elements on a page. It's about guiding someone through an experience. What do they feel first? What comes next? When do they pause? When do they act?

WKALAN layouts are editorial — they reference magazines, books, and photobooks more than web conventions. Each layout follows a deliberate narrative arc.

---

## The Six Sacred Layouts

### 1. The Hero
*Purpose:* First impression. Brand statement. Emotional hook.

```
┌──────────────────────────────┐
│                              │
│      Full-bleed image        │
│      or dark canvas          │
│                              │
│      MASSIVE HEADLINE        │
│      Subtitle below          │
│                              │
│      ↓ Scroll indicator      │
└──────────────────────────────┘
```

- Full viewport height
- Typography-led or image-led (never both competing)
- One CTA maximum
- Gradient fade at bottom edge → next section

### 2. The Statement
*Purpose:* Declare a belief. Set the philosophical tone.

```
┌──────────────────────────────┐
│                              │
│   SECTION LABEL (small)      │
│   ───── (thin line)          │
│                              │
│   Large quote or statement   │
│   in serif. Centered or      │
│   left-aligned.              │
│                              │
│   Supporting body text       │
│   below, narrower width.     │
│                              │
└──────────────────────────────┘
```

- Generous vertical space
- Narrow reading width (max 720px)
- Label → line → quote → body rhythm

### 3. The Grid
*Purpose:* Present multiple items with equal weight.

```
┌──────────────────────────────┐
│   SECTION LABEL              │
│   ─────                      │
│                              │
│   ┌─────┐ ┌─────┐ ┌─────┐  │
│   │     │ │     │ │     │  │
│   └─────┘ └─────┘ └─────┘  │
│   ┌─────┐ ┌─────┐ ┌─────┐  │
│   │     │ │     │ │     │  │
│   └─────┘ └─────┘ └─────┘  │
└──────────────────────────────┘
```

- 2, 3, or 4 columns
- Equal card sizes → equal importance
- Staggered reveal animation
- Consistent gap throughout

### 4. The Timeline
*Purpose:* Show progression. Tell a journey.

```
┌──────────────────────────────┐
│         │                    │
│  2020 ──●── Event left       │
│         │                    │
│         │     Event right ──●── 2021
│         │                    │
│  2022 ──●── Event left       │
│         │                    │
└──────────────────────────────┘
```

- Central vertical line (gold or subtle)
- Alternating left/right entries
- Year markers as anchors
- Scroll-triggered reveals from alternating sides

### 5. The Split
*Purpose:* Compare. Contrast. Show two sides.

```
┌──────────────────────────────┐
│                    │         │
│   Text content     │  Image  │
│   Left-aligned     │  Right  │
│                    │         │
└──────────────────────────────┘
```

- 50/50 or 60/40 split
- Text on one side, image on the other
- Reverses on alternate rows
- Collapses to stack on mobile

### 6. The Close
*Purpose:* End the journey. Call to action.

```
┌──────────────────────────────┐
│                              │
│      Final statement         │
│      or question             │
│                              │
│      [ CTA Button ]          │
│                              │
│      Contact links below     │
│                              │
└──────────────────────────────┘
```

- Centered, tight, focused
- One clear action
- Generous whitespace before and after
- Feels like: the last page of a book

---

## Layout Rules

### 1. Every page has one hero
Don't dilute the first impression. One hero section per page. One statement that sets the tone.

### 2. Sections alternate rhythm
Light → slightly darker → light → slightly darker. Or: full-bleed → contained → full-bleed → contained. Rhythm prevents monotony.

### 3. Reading width is sacred
Body text never exceeds 720px. Headlines can go wider, but sentences need boundaries.

### 4. The close is not an afterthought
The last thing someone sees is what they remember. Design the close with as much care as the open.

---

*Layout is storytelling in space. Every section is a sentence. Every page is a chapter.*
